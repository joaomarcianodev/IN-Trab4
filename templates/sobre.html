Sobre o Projeto

Este projeto de Recomendação de Alimentos foi desenvolvido como uma iniciativa para explorar e aplicar técnicas de análise de dados e aprendizado de máquina. O objetivo principal é agrupar alimentos com base em suas informações nutricionais, permitindo a identificação de grupos alimentares similares e a geração de insights valiosos para dietas e planos alimentares.

Utilizamos a plataforma KNIME para o pré-processamento, modelagem e análise dos dados. Foram testados diversos algoritmos de clusterização para encontrar a melhor estrutura de agrupamento para a base de dados selecionada.
